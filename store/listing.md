## Short description (132 chars max — used as the subtitle on both stores)

Brings back the Add to Queue and Watch Later buttons YouTube removed from video thumbnails.

---

## Long description

YouTube quietly removed the Add to Queue and Watch Later hover buttons from video thumbnails. YT Restore brings them back — exactly where you'd expect them.

**What it adds:**

- A Queue button and a Watch Later button appear on every video thumbnail when you hover over it, positioned at the top-right corner
- Works on the YouTube home feed, search results, sidebar recommendations, and channel pages
- Shorts get a Queue button only (Watch Later is not supported by YouTube for Shorts)

**How it works:**

The buttons use YouTube's own native actions — the same ones triggered when you use the three-dot context menu. There's no custom API, no workarounds, and no risk of your queue or Watch Later list behaving differently than it would normally.

**Privacy:**

This extension collects no data whatsoever. It does not track you, phone home, or interact with any server. It only runs on YouTube pages and only does what you ask it to do.
