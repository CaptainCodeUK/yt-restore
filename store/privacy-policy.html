Privacy Policy

YT Restore — Queue & Watch Later
Last updated: March 2026

Summary

YT Restore does not collect, store, transmit, or share any personal data or browsing information. Full stop.

What the extension does

YT Restore is a browser extension that adds Queue and Watch Later buttons back to YouTube video thumbnails. It operates entirely within your browser and interacts only with YouTube's own on-page interface to perform actions you explicitly request (adding a video to your queue or Watch Later list).

Data collection

This extension:

  • Does not collect any personal information.
  • Does not track your browsing history or watch history.
  • Does not transmit any data to any server — ours or anyone else's.
  • Does not use cookies, analytics, or any third-party services.
  • Does not store anything locally beyond what the browser requires to run the extension itself.

Permissions

The extension requests access to youtube.com solely to inject the Queue and Watch Later buttons into the YouTube interface. No data from these pages is read, recorded, or transmitted.

Third parties

This extension has no relationship with YouTube or Google. When you use the Queue or Watch Later buttons, those actions are performed using YouTube's own on-page mechanisms — the same actions that occur when you use YouTube's native interface.

Changes to this policy

If the extension ever changes in a way that affects privacy, this policy will be updated and the extension version number will be incremented.

Contact

If you have questions about this policy, please open an issue on the extension's GitHub repository.

YT Restore is an independent open-source project and is not affiliated with YouTube or Google.
